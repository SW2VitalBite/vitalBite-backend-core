# Formularios

## Objetivo

Registrar y actualizar información mediante formularios reactivos.

## Formularios principales

- Paciente.
- Cita.
- Medida corporal.
- Composición corporal.
- Dieta.
- Alimento.
- Receta.
- Plantilla de dieta.
- Antropometría avanzada.
- Meta del paciente.

## Validaciones

- Requeridos.
- Rangos numéricos.
- Fechas válidas.
- Formato de email.
- Fuente única para ítems de dieta o plantilla.
- Unidades correctas para antropometría.

## Reglas

- Validar en frontend antes de enviar.
- El backend conserva validación final.
- Mostrar errores GraphQL en lenguaje claro.
