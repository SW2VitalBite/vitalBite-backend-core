# Gráficos

## Objetivo

Visualizar evolución e indicadores BI.

## Gráficos esperados

- Evolución de peso.
- Evolución de IMC.
- Composición corporal.
- Adherencia diaria.
- Cumplimiento de dieta.
- Citas completadas/canceladas.
- Riesgo nutricional.
- Segmentación K-means.
- Somatocarta.

## Reglas

- Los datos vienen del Core GraphQL.
- Los gráficos deben manejar estado vacío.
- Somatocarta debe usar coordenadas documentadas por el Core.
- Indicadores financieros solo se muestran con permisos.
