# Modales

## Objetivo

Confirmar acciones o editar información sin perder contexto.

## Usos esperados

- Confirmar eliminación lógica.
- Cancelar cita.
- Reprogramar cita.
- Activar o cancelar dieta.
- Confirmar generación de PDF.
- Completar meta.
- Mostrar detalle breve de error.

## Reglas

- Acciones críticas deben pedir confirmación.
- No usar modales para flujos largos si requieren pantallas completas.
- Mostrar estado de carga durante mutations.
- Cerrar modal solo cuando la operación termine correctamente o el usuario cancele.
