# Layout principal

## Objetivo

Definir la estructura visual base para pantallas internas.

## Elementos

- Sidebar de navegación.
- Topbar con usuario autenticado.
- Área principal de contenido.
- Breadcrumb o título de sección.
- Contenedor responsive.

## Reglas

- El layout interno solo se muestra si el usuario está autenticado.
- El login no usa layout administrativo.
- Las pantallas deben mantener consistencia visual y navegación clara.
- El contenido debe adaptarse a escritorio y tablet.
