# Comandos útiles

## Desarrollo

```bash
ng serve
```

## Compilación

```bash
ng build
```

## Pruebas

```bash
ng test
```

## Lint

```bash
ng lint
```

## Generación sugerida

```bash
ng generate component features/patients/pages/patients-list
ng generate service core/graphql/graphql
ng generate guard core/guards/auth
```

Los comandos exactos pueden ajustarse según el proyecto Angular real.
