# Requisitos

## Requisitos técnicos

- Node.js LTS.
- pnpm 10+.
- Angular CLI.
- Navegador moderno.
- Acceso al endpoint GraphQL del Core NestJS.
- Variables de entorno para URL del Core y configuración de sesión.

## Dependencias esperadas

- Angular.
- Apollo Angular o cliente GraphQL equivalente.
- RxJS.
- Formularios reactivos de Angular.
- Librería de gráficos para dashboard BI.
- Librería de iconos o componentes UI definida por el equipo.

## Servicios requeridos

- Core NestJS + GraphQL.
- Servicio Documental para URLs prefirmadas a través del Core.
- Microservicio .NET para pagos y suscripciones consultado por el Core.
- FastAPI para resultados IA/ML/DL consultados por el Core.
