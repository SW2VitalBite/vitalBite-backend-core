# Instalación

## Instalación base

```bash
npm install
```

## Configuración inicial

1. Crear archivo de variables de entorno del frontend.
2. Configurar URL del endpoint GraphQL del Core.
3. Configurar nombre de aplicación y entorno.
4. Configurar política de almacenamiento del token.
5. Instalar dependencias de GraphQL, UI y gráficos.

## Ejecución local

```bash
ng serve
```

## Verificación

La aplicación debe iniciar sin errores, mostrar `/login` y poder conectarse al endpoint GraphQL configurado.
