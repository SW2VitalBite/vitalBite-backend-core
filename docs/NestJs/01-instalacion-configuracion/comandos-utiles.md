# Comandos útiles

## Desarrollo

```bash
npm run start:dev
```

## Compilación

```bash
npm run build
```

## Pruebas

```bash
npm run test
```

## Pruebas end-to-end

```bash
npm run test:e2e
```

## Lint

```bash
npm run lint
```

## Formato

```bash
npm run format
```

Los comandos exactos pueden ajustarse según el `package.json` real del microservicio.
