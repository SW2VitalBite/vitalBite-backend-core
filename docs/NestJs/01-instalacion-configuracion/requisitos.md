# Requisitos

## Requisitos técnicos

- Node.js LTS.
- npm o yarn.
- NestJS CLI.
- PostgreSQL o acceso a Supabase.
- Cliente GraphQL para pruebas.
- Git.

## Servicios requeridos

- PostgreSQL/Supabase para datos del Core.
- Microservicio .NET para validar planes y suscripciones.
- Microservicio Spring Boot para documentos y auditoría.
- Microservicio FastAPI para IA/ML/DL.

## Conocimientos necesarios

- TypeScript.
- NestJS.
- GraphQL.
- PostgreSQL.
- Autenticación JWT.
- Arquitectura modular y microservicios.
